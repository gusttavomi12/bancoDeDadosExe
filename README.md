 # 📚 Conceitos Básicos de Banco de Dados

## 1. O que é banco de dados?
Banco de dados é um conjunto organizado de informações que podem ser armazenadas, acessadas e manipuladas de forma eficiente.

---

## 2. O que é SGBD e qual a vantagem?
**SGBD (Sistema de Gerenciamento de Banco de Dados)** é um software que permite criar, gerenciar e manipular bancos de dados.

**Vantagens:**
- Organização dos dados
- Segurança das informações
- Facilidade de acesso e atualização
- Controle de usuários

---

## 3. O que é um DBA e qual sua responsabilidade?
**DBA (Administrador de Banco de Dados)** é o profissional responsável por gerenciar o banco de dados.

**Responsabilidades:**
- Garantir segurança dos dados
- Fazer backup e recuperação
- Monitorar desempenho
- Controlar acessos

---

## 4. Conceito de modelo lógico e modelo físico

- **Modelo Lógico:** representa a estrutura do banco de dados de forma abstrata (tabelas, relacionamentos), sem se preocupar com a implementação.

- **Modelo Físico:** mostra como os dados serão armazenados de fato no sistema (tipos de dados, índices, armazenamento).
